Studio d'enregistrement

Réaliser de façon simple et autonome, vos propres supports de cours audio et/ou vidéo. L'utilisation du navigateur web Chrome est préconisée.

Pour cela, trois modes d’enregistrement sont proposés :

  1. Audio : enregistrez une retranscription audio de votre cours
  2. Vidéo + Audio : réalisez une capsule vidéo de votre cours à partir de votre webcam
  3. Support de cours + Audio : faites une capture d'écran vidéo (screencast) pour accompagner un support de cours (PPT, PDF) d’un commentaire audio

Une fois votre captation terminée, exportez votre travail sur votre ordinateur. Vous pourrez ensuite héberger ce support de cours dans POD, la webtv de l’UGA, pour ensuite l’intégrer sur la plateforme pédagogique de votre cours.
